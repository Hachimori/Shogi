# 入力ファイル
- フォーマット
```
[先手か後手か (SENTE or GOTE)]
[盤上にある先手の駒の数]
先手の駒の種類_1 行_1 列_1 (成っているか (HUNARI or NARI)_1)
先手の駒の種類_2 行_2 列_2 (成っているか (HUNARI or NARI)_2)
...
[盤上にある後手の駒の数]
後手の駒の種類_1 行_1 列_1 (成っているか (HUNARI or NARI)_1)
後手の駒の種類_2 行_2 列_2 (成っているか (HUNARI or NARI)_2)
...
[先手の持駒の数]
先手の持駒の種類_1
先手の持駒の種類_2
...
[後手の持駒の数]
後手の持駒の種類_1
後手の持駒の種類_2
...
```

- 行は *上* から順に 0, 1, 2, ... と数える
- 列は *右* から順に 0, 1, 2, ... と数える

- 駒の種類は、以下のいずれか
  - HU (歩)
  - KYOSYA (香)
  - KEIMA (桂)
  - GIN (銀)
  - KIN (金)
  - HISYA (飛)
  - KAKU (角)
  - GYOKU (玉。先手後手によらずに GYOKU と表記する)
